- create `Chat/shared/Chat.graphql`
	 - chat has many messages

**Chat.graphql**
```typescript
type  Chat {
	id: Int!
	messages: [Message]!
	participants: [User]!
	createdAt: String!
	updatedAt: String
}
```
- create `Chat/shared/Message.graphql`
	- message has one chat.

**Message.graphql**
```typescript
type  Message {
	id: Int!
	text: String!
	chat: Chat!
	user: User!
	createdAt: String!
	updatedAt: String
}
```
